# Setting Up a Next.js Project with Shadcn Tailwind

In this guide, we'll walk through the process of setting up a Next.js project with Shadcn Tailwind and other essential installations. Follow the steps below to get started.

## Step 1: Install Tailwind CSS

First, install Tailwind CSS and its dependencies:

```bash copy showLineNumbers
npm install tailwindcss
npm install -D tailwindcss postcss autoprefixer
```
